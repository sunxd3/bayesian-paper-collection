# Hierarchical Bayesian Brain Cell Counts

Hierarchical Poisson models with horseshoe shrinkage and zero-inflation for multiregion brain cell counts.

## Citation

Dimmock S, Exley BMS, Moore G, Menage L, Delogu A, Schultz SR, Warburton EC, Houghton CJ, O'Donnell C (2025). Hierarchical Bayesian modeling of multiregion brain cell count data. *eLife* 14:RP102391.
