# Rat Multisensory Psychophysics

Hierarchical probit psychometric model for rat visual classification under multisensory conditions.

## Citation

Zanzi M, Rinaldi FG, Fornasaro S, Piasini E, Zoccolan D (2025). Seeing what you hear: Compression of rat visual perceptual space by task-irrelevant sounds. *PLOS Computational Biology* 21(10): e1013608.

## Links

- **Paper**: https://doi.org/10.1371/journal.pcbi.1013608
- **Data/Code**: https://zenodo.org/records/17280352

## Metadata

- **PPL**: PyMC v5.2
- **Language**: Python
- **Published**: Oct 2025
- **Scale**: ~78k Bernoulli trials, 10 rats, 5 model variants compared via PSIS-LOO
